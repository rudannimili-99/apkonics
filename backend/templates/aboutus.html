About Apkonic

📌 Project Introduction

Apkonic is a web-based security system developed to detect fraudulent APK files and smishing (SMS phishing) attacks. The platform is designed to protect users from malicious applications and scam messages that can compromise personal and financial information.

⚠ Problem Statement

With the rapid growth of digital banking and online services, cyber fraud has significantly increased. Many users unknowingly install malicious APK files or click on phishing SMS links, resulting in data theft and financial loss. There is a need for a simple and accessible system that helps users identify such threats before damage occurs.

🎯 Our Mission

Our mission is to create a safer digital environment by providing a lightweight, fast, and user-friendly detection system that identifies suspicious applications and scam messages in real-time.

🛠 Technologies Used

• Python (Flask Framework)
• HTML5 & CSS3
• JavaScript (Fetch API)
• Keyword-based phishing detection logic
• File validation and size analysis

⚙ How It Works

1. User uploads an APK file or enters an SMS message.
2. The system analyzes file properties and phishing keywords.
3. Suspicious indicators are detected.
4. The system displays a clear SAFE or WARNING result instantly.

🚀 Future Enhancements

• Integration with Machine Learning models for advanced detection
• Real-time database of malicious APK signatures
• Cloud deployment and API integration
• Admin dashboard for monitoring threats
